# 🚗 Car Pricer

Market valuation tool for automotive resale professionals.  
Aggregates European market data (AutoScout24, LaCentrale, LeBonCoin) into a local SQLite database and enables real-time querying of a vehicle's market value.

---

## Features

- **Local database** — 100,000+ EU listings imported from AutoScout24
- **Full market stats** — min, P10, P25, median, P75, P90, max per vehicle and mileage bracket
- **Deal indicator** — buy threshold (P25) and overpriced threshold (P75) automatically calculated
- **Smart search** — fuzzy matching on make and model (typo-tolerant)
- **Advanced filters** — year, mileage ± margin, fuel type, transmission, country
- **Multi-source** — architecture ready for LaCentrale and LeBonCoin
- **Multi-country** — data from FR, DE, IT, ES, BE, NL, AT

### Requirements
- Python 3.11+
- macOS / Linux

```bash

# 2. Run the one-shot setup
```

## Usage

### Step 1 — Import data

Download the AutoScout24 dataset and place it in `data/` :

```bash
# Dry run (no data written to database)
python scripts/2_import_autoscout24.py --dry-run

# Full import (~50,000 EU listings)
python scripts/2_import_autoscout24.py

# Filter by a single country
python scripts/2_import_autoscout24.py --country FR

# Custom CSV path
python scripts/2_import_autoscout24.py --csv data/my_file.csv
```

## Database

### Tables

| Table | Description |
|---|---|
| `vehicles` | Vehicle reference — make, model, trim, year, specs |
| `listings` | Imported listings — price, mileage, country, seller, vehicle history |
| `reference_prices` | Reference valuations (Argus, LaCentrale Pro) |
| `market_stats` | Calculated stats per vehicle / mileage bracket / country |
| `portfolio` | Purchased and sold vehicles — margin tracking |

### Views

| View | Description |
|---|---|
| `v_vehicle_market` | Vehicle + latest market stats |
| `v_portfolio_valuation` | Portfolio + estimated potential gain |

### Useful queries in DBeaver

```sql
-- Market stats for a specific model
SELECT * FROM v_vehicle_market
WHERE make = 'Volkswagen' AND model = 'Golf'
ORDER BY production_year DESC;

-- Best deals right now (price < P25)
SELECT l.price, l.mileage_km, l.country_code, v.make, v.model,
       v.production_year, ms.price_p25
FROM listings l
JOIN vehicles v ON v.id = l.vehicle_id
JOIN market_stats ms ON ms.vehicle_id = l.vehicle_id AND ms.country_code = 'ALL'
WHERE l.price < ms.price_p25
  AND l.is_active = 1
ORDER BY (ms.price_p25 - l.price) DESC
LIMIT 20;

-- Listing count by country
SELECT country_code, COUNT(*) as nb_listings
FROM listings
GROUP BY country_code
ORDER BY nb_listings DESC;

-- Wipe all tables (full reset)
PRAGMA foreign_keys = OFF;
DELETE FROM market_stats;
DELETE FROM listings;
DELETE FROM reference_prices;
DELETE FROM portfolio;
DELETE FROM vehicles;
DELETE FROM sqlite_sequence WHERE name IN
    ('market_stats','listings','reference_prices','portfolio','vehicles');
PRAGMA foreign_keys = ON;
```

## Configuration

All settings are centralised in `config/settings.py` :

```python
GBP_TO_EUR   = 1.17      # GBP to EUR conversion rate (UK dataset)
MILES_TO_KM  = 1.60934   # Miles to km conversion

MILEAGE_BRACKETS = [     # Mileage brackets for stats calculation
    (0,       50_000),
    (50_000,  100_000),
    (100_000, 150_000),
    (150_000, 200_000),
    (200_000, 999_999),
]
```
